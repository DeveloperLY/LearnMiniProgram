### 代码如何运行

> 代码中的每个知识点使用tag进行管理，所以查看对应的知识点需要切换tag

1.将代码clone到本地（注意这里不能直接download）

```shell
git clone https://github.com/DeveloperLY/LearnMiniProgram.git
```

2.进入目录，查看某个知识点，切换到对应的tag

```shell
git checkout 11-项目实战演练
```
